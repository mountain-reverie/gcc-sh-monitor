# CSiBE per-subproject licensing

All subprojects below are GPL-compatible (BSD-like, MIT-like, zlib,
IJG, or (L)GPL). This file is a quick inventory; consult each
subproject's own LICENSE/COPYING file for authoritative terms.

The CSiBE framework itself is covered by the top-level `COPYING` file
(GPLv2).

## bzip2-1.0.2
License: bzip2 (BSD-like). See `src/bzip2-1.0.2/LICENSE`.

## cg_compiler_opensrc
License: NVIDIA-derived permissive. See `src/cg_compiler_opensrc/LICENSE`
and `src/cg_compiler_opensrc/License.txt`.

## compiler
License: unclear — no LICENSE/COPYING file in the subproject; only
`readme.txt`. This is a small "vc compiler / vas assembler / VAM
interpreter" demo distribution included by the CSiBE authors.
Treated as redistributable under the parent CSiBE corpus terms
(GPLv2, top-level `COPYING`).
